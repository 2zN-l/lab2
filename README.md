# ЗАДАНИЕ 1
#### Задание: Иван составляет 5-буквенные коды из букв И, В, А, Н. Буквы в коде могут повторяться, использовать все буквы не обязательно, но букву И нужно использовать хотя бы один раз. Сколько различных кодов может составить Иван?
<br>
Для решения задачи использовалась библиотека itertools. С помощью функции product были сгенерированы все возможные комбинации длиной 5 из букв 'И', 'В', 'А', 'Н'. В цикле проверялось условие, что буква 'И' встречается хотя бы один раз. Подсчитывалось количество подходящих комбинаций. Результат был выведен на консоль.

# ЗАДАНИЕ 2

#### Задание: Значение арифметического выражения $`7*512^{120} - 6*64^{100} + 8^{210} - 255`$ записали в системе счисления с основанием 8. Сколько цифр 0 содержится в этой записи?
<br>
Было вычислено значение выражения с помощью арифметических операций Python. Затем реализован алгоритм перевода числа в систему счисления с основанием 8 путем последовательного деления на 8 и записи остатков. Полученная строка была перевернута для получения правильной записи числа. С помощью метода count подсчитано количество нулей в восьмеричной записи. Результат выведен на консоль.

# ЗАДАНИЕ 3

#### Задание: Найдите среди целых чисел, принадлежащих числовому отрезку [84052; 84130], число, имеющее максимальное количество различных натуральных делителей, если таких чисел несколько  — найдите минимальное из них. Выведите на экран количество делителей такого числа и само число.
<br>
Был организован перебор всех чисел в заданном диапазоне. Для каждого числа подсчитывалось количество натуральных делителей (включая 1 и само число). В процессе перебора отслеживалось максимальное найденное количество делителей и соответствующее число. Если находилось число с таким же количеством делителей, выбиралось минимальное. Результат был выведен на консоль в формате: количество делителей и само число.
